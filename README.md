# Unrealities 影想 — AI 動畫工作室（範本網站）

用 Claude Code 製作的中文版單頁形象網站：會動的主視覺、作品 / 服務 / 製作流程 / 關於、聯絡表單、手機響應式、滑入動畫。

## 本機預覽

直接用瀏覽器打開 `index.html` 即可 —— 單一檔、不需要伺服器。

## 換上角色影片

把白底角色影片放進 `assets/`，命名為 `character1.mp4`（左）、`character2.mp4`（右），
重新整理頁面就會自動把示意角色換成你的影片（框成圓角人像卡）。沒有檔案時保持示意角色，不會壞。
